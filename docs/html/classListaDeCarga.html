Detailed Description

Lista doblemente enlazada para almacenar el mensaje decodificado.

Esta lista almacena los caracteres decodificados en el orden en que fueron procesados. Al final del proceso, contiene el mensaje oculto completo.

Características:

  • Inserción eficiente al final (O(1))
  • Navegación bidireccional
  • Preserva el orden de llegada de los datos
